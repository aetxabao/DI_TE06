FAQ: Preguntas más frecuentes

  1. Es necesario disponer de una base de datos SQL Server?
    • Sí, así es. SQL Server debe estar funcionando con la base de datos GRASUPINF
    • Compruebe que el servicio está arrancado, puede acceder a los servicios ejecutando services.msc
    • SQL Server Management Studio le puede ayudar a comprobar la instalación
  2. Es necesario disponer tener instalado Crystal Reports?
    • Sí, sólo para los informes hecho con Crystal Reports
    • Créese una cuenta y descarguese el software de Crystal Reports
  3. Necesito tener Visual Studio de Microsoft?
    • Sólo para para modificar el programa, no para ejecutarlo.
